# Telegram-бот: напоминания, привычки, сон и трекер воды

Удобный бот для личной продуктивности с управлением через кнопки. Время по умолчанию — Москва (Europe/Moscow), все данные разделены по пользователям.

## Главное меню

- 🎯 Привычки — сразу список с кнопками отметки
- 💧 Вода — быстрое добавление и прогресс
- 📅 Сегодня — дашборд дня
- 😴 Сон — режим сна, стрик, напоминания
- ➕ Напоминание — добавить напоминание
- 📋 Мои напоминания — список
- 🗑 Удалить напоминание
- ❓ Помощь

## Возможности

### Напоминания
- Быстрые кнопки времени: 5/10/30 мин, 2/6/12 ч
- Или своё время в формате `ДД.ММ.ГГГГ ЧЧ:ММ`
- Разовые, ежедневные, еженедельные, ежемесячные
- Все напоминания по Москве
- Сохраняются в SQLite и восстанавливаются после перезапуска

### Привычки
- 👍 Полезные — отмечай выполнение кнопкой ✅
- 🚫 Вредные — отмечай сдержанность кнопкой 🔥 Держусь
- 🔥 Отдельный стрик для каждой привычки
- 📊 Статистика за 30 дней
- ⏰ Напоминания 3 раза в день (по умолчанию 09:00, 13:00, 21:00)
- ⚙️ Время напоминаний можно изменить или выключить

### Сон
- 😴 Задай время отбоя и подъёма
- 🔥 Стрик сна — счётчик дней подряд с вовремя отбоя и подъёма
- ⏰ Напоминания вечером (за 15 минут до сна) и утром (время подъёма)
- ✅/❌ Быстрые кнопки отметки прямо в напоминаниях
- Режим и напоминания можно включать/выключать

### Вода
- 💧 Быстрые кнопки: +200/+300/+500 мл
- Прогресс-бар и остаток до цели
- Статистика за 7 дней
- Настраиваемая дневная цель

### Дашборд «Сегодня»
- Все привычки на сегодня
- Выпитая вода
- Режим сна и стрик
- Ближайшие напоминания

## Деплой на BotHost.ru

1. Получите токен бота у [@BotFather](https://t.me/BotFather).
2. На BotHost.ru укажите:
   - **Ссылка на репозиторий**: `https://github.com/ВАШ_НИКНЕЙМ/reminder-bot`
   - **Файл запуска**: `main.py`
   - **Переменные окружения**:
     - `BOT_TOKEN` — токен от @BotFather
     - `DEFAULT_TIMEZONE` — часовой пояс (по умолчанию `Europe/Moscow`)
3. Запустите бота.

## Локальный запуск

```bash
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
export BOT_TOKEN="ваш_токен"
python main.py
```
